# Hand Mouse Control

Control your computer mouse using hand gestures and webcam.

---

## Features

- Move mouse cursor using index finger movement
- Left click by bringing index and middle finger together
- Double click using peace sign gesture
- Lock mouse with fist gesture
- Smooth cursor movement with smoothing algorithm
- Real-time hand tracking with 21 landmarks

---

## Technologies Used

| Technology | Purpose |
|------------|---------|
| Python 3.8+ | Programming language |
| OpenCV | Camera capture and image processing |
| MediaPipe | Hand landmark detection |
| PyAutoGUI | Mouse control |
| NumPy | Mathematical operations |

---

## Requirements

- Python 3.8 or higher
- Webcam
- Windows / macOS / Linux
